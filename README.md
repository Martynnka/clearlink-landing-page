# Clearlink – Landing Page

**Clearlink** is a Lithuanian project focused on improving online safety.  
This repository contains the **landing page** for the Clearlink idea – a concept tool designed to help users **check website security before clicking a link**.

The goal of Clearlink is to reduce phishing attacks, scam websites, and malicious links by giving users a clearer understanding of potential risks.

---

## About the Project

Clearlink is a **Lithuanian cybersecurity-focused concept project** that aims to:

- Detect suspicious website indicators  
- Analyze HTTPS and connection security  
- Identify unusual domain structures  
- Detect redirects  
- Provide a simplified risk evaluation system  

> ⚠️ Note: This repository contains the **frontend landing page only** (HTML + CSS). It does not include backend logic or real-time security scanning functionality.

---

## Pages Included

- `index.html` – Main landing page with security check concept and explanation
